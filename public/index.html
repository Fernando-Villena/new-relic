Herramienta de Alertas New Relic

Policy Alertas Configuradas

Ingresa un Policy ID para obtener información detallada de las alertas asociadas a esa política. Podrás exportarlas directamente a un archivo Excel. 


  // Cómo usar la aplicación:

  1. Escribe el Policy ID en el campo de texto.  
     (Ejemplo: 1234567)

  2. Haz clic en 
     "Exportar alertas".

  3. La app consultará la 
     API de New Relic y, si existen alertas,  
     descargará un archivo "alerts.xlsx".

  4. El archivo Excel contendrá:  
     - ID, Nombre y Descripción  
     - Estado (habilitada o no)  
     - Tipo de condición  
     - Runbook asociado  
     - Consulta NRQL  
     - Detalles de umbral, operador, prioridad,  
       duración y ocurrencias

  5. Si la Policy ID no tiene alertas asociadas,  
     se mostrará un aviso en pantalla  
     y no se descargará ningún archivo.
    

  // Resumen de alertas configuradas:

  Este panel permite exportar todas las alertas existentes en el sistema
  a un archivo Excel haciendo clic en 
  "Exportar todas las alertas".

  El archivo contendrá para cada alerta:
    - ID, Nombre y Descripción
    - Estado (habilitada o no)
    - Tipo de condición
    - Runbook asociado
    - Consulta NRQL
    - Detalles de umbral, operador, prioridad, duración y ocurrencias

  Si no hay alertas configuradas, se mostrará un aviso en pantalla.
